# Brain — Инициализация проекта

**Brain** — проект документации на базе [Zensical](https://zensical.org), поддерживаемый организацией [bitplain](https://github.com/bitplain).

## О проекте

Репозиторий `bitplain/brain` содержит базу знаний и техническую документацию, собранную с помощью генератора статических сайтов Zensical.

## Структура проекта

```
brain/
├── docs/          # Исходные файлы документации (Markdown)
│   ├── index.md   # Главная страница
│   ├── init.md    # Описание проекта (этот файл)
│   └── markdown.md
├── zensical.toml  # Конфигурация Zensical
└── .github/       # CI/CD workflows
```

## Быстрый старт

| Команда              | Описание                          |
|----------------------|-----------------------------------|
| `zensical serve`     | Запустить локальный сервер        |
| `zensical build`     | Собрать статический сайт          |
| `zensical new`       | Создать новый проект              |

## Технологии

- **[Zensical](https://zensical.org)** — генератор документации с поддержкой тем, навигации, поиска и многого другого.
- **Markdown** — формат исходных файлов документации.
- **GitHub Actions** — автоматизация сборки и публикации.
