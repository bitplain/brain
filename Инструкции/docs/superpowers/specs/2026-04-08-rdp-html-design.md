# Дизайн HTML-памятки по удаленному рабочему столу

**Дата:** 2026-04-08

## Цель

Собрать одностраничную HTML-инструкцию для внутренних пользователей, которая в понятной форме объясняет разницу между локальным и удаленным рабочим столом и показывает порядок входа в удаленную сессию.

## Формат

- Одна HTML-страница без внешних зависимостей.
- Нейтральный деловой стиль.
- Спокойная корпоративная палитра: глубокий синий, светло-серый фон, приглушенные синие и зеленые акценты.
- Встроенные стили в одном файле для простого открытия локально.

## Структура страницы

1. Верхний hero-блок с заголовком и кратким пояснением.
2. Выделенный warning-блок с ключевой мыслью:
   ярлык нужно искать на локальном рабочем столе, а не в уже открытой удаленной сессии.
3. Две карточки-сравнения:
   локальный рабочий стол и удаленный рабочий стол.
4. Шаг 1:
   поиск ярлыка удаленного рабочего стола на локальном экране с использованием иконки.
5. Шаг 2:
   окно ввода учетных данных с использованием скриншота и двумя поясняющими блоками:
   первый запуск и повторный запуск.
6. Финальный блок с кратким алгоритмом действий.

## Контентные требования

- Обязательно использовать формулировку учетной записи:
  `inf.co.fi\пользователь`
- Ясно разделить первый запуск и повторный запуск.
- Показать оба предоставленных изображения:
  иконку ярлыка и окно входа.
- Избегать перегруженного технического языка.

## Визуальный подход

- Заголовки крупные, строгие и контрастные.
- Карточки с мягкими тенями и тонкими рамками.
- Акценты только на важных местах, без избыточной декоративности.
- Страница должна одинаково хорошо читаться на широком экране и на ноутбуке.

## Доступность и удобство

- Достаточный цветовой контраст.
- Понятная иерархия заголовков.
- Адаптивная верстка без горизонтальной прокрутки.
- Печатный вид без лишнего интерфейсного шума.

## Ограничения

- Без фреймворков и сборщиков.
- Без внешних шрифтов и CDN.
- Локальные изображения подключаются по относительным путям из этой же папки.
