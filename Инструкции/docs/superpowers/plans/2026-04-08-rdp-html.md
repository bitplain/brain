# RDP HTML Guide Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a one-page local HTML guide in a neutral corporate style that explains how to connect to Remote Desktop.

**Architecture:** Use a single standalone HTML file with embedded CSS and semantic sections. Reuse the existing local PNG assets for the shortcut icon and login window so the page opens directly from disk without any build step.

**Tech Stack:** HTML5, CSS3, local image assets
